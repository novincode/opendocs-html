🦀 Your First Rust Docs Page: Hello, World! Style

So you just started learning Rust and want your code to look pro? Let’s write a Hello, World! that’s not just code—it’s documented code.

✨ Step 1: Make a New Project

Open your terminal and type:

bash
cargo new hello_docs
cd hello_docs

✍️ Step 2: Add Some Friendly Docs

Open src/lib.rs and use this:

rust
1//! Welcome to hello_docsCrate-level doc comment.
2//! Prints Hello, world!
3
4/// Prints hello
5pub fn say_hello() {
6    println!("Hello, world!");
7}

🔍 Step 3: See the Magic

bash
cargo doc --open

✅ Bonus: Make It Run

Edit src/main.rs:

rust
fn main() {
    hello_docs::say_hello();
}

And in Cargo.toml:

toml
[lib]
path = "src/lib.rs"

💡 Tips for Writing Docs

  • Use //! for crate docs
  • Use /// for functions
  • Show examples with triple backticks
  • Be clear and friendly
